Docstring

The Folk docstring consists of:

  • A title line
  • --- (three underscores) to separate the title line from the summary
  • A one-line summary
  • Supporting documentation
  • --- (three underscores) to separate the supporting documentation from the example string
  • The example When/Claim string
  • --- (three underscores) Final three dashes, ends the docstring

e.g.:

# This is the title
    # ---
    # This is my smol summary
    # This is the supporting documentation. I can write anything I want in here.
    # The parser will look for the next three undersores so I can put ¯\_(ツ)_/¯ in here too.
    # ---
    # Claim name is "Euclid"
    # Claim name is "George Pólya"
    # ---
    When /someone/ claims name is /name/ {
        Claim /someone/ is a mathematician
    }
