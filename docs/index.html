Virtual Programs

Virtual programs are `.folk` programs that run automatically when a Folk system boots. These programs define the core capabilities of the system (e.g. connecting to a webcam, drawing text, displaying shapes, connecting keysboards, etc.) and define what `Wishes` and `Claims` are built-in to Folk.
