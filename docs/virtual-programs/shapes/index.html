Shapes

Drawing a shape

This wish allows you to draw a shape with a given number of `sides`, `center`, `radius`, `color`, and a `filled` value. 

sides 2 => line
sides 3 => triangle
sides 4 => square
sides 5 => pentagon
sides 6 => hextagon
sides 7 => septagon
sides 8 => octagon
sides 9 => nonagon

Example usage

Wish to draw a shape with sides /s/ center /c/ radius /r/ color /color/ filled /filled?/
